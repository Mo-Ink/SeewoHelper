# SeewoHelper

SeewoHelper 是一个为**解决管理员在 Seewo 一体机上操作不便**而诞生的**教室多媒体辅助程序**（当然它**也适用于其他的触摸大屏设备**），**你可能需要物理接触一体机才能部署它**。

[查看开发文档](docs/开发文档.md) 

## 特性

- 可设置开机自动启动
- 可自动更新主程序
- 可配置性

## 功能

- 桌面整理
