# SeewoHelper

SeewoHelper 是一个**为解决Seewo一体机屏幕过大而导致的操作不便**诞生的辅助程序（当然它**也适用于其他的触摸大屏PC设备**）。

## 特性

- 可设置开机自动启动
- 可自动更新主程序
- 可配置性

## 目标功能（暂定）

- 快捷电源控制
- 定时功能
- 一键关闭/重启WPS或office进程
- 课件自动整理
- [充当RemoteExecutor的ClientLauncher](https://github.com/zi-jing/RemoteExecutor)
- 高危指令可密码上锁
